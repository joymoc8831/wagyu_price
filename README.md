# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

#### **ローカルリポジトリ**は自分のPC内での保管
#### **リモートリポジトリ**はチームで共有している保管場所


## プッシュとマージの違いは何でしょうか？

#### branchの変更履歴を結合するのがmerge、branchをリモートリポジトリへアップロードするのがpush


## コミットとプッシュの違い

#### commitは変更した内容をローカルリポジトリへ一時保存することで、pushはcommitで保存したものをリモートリポジトリへアップロードすること

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

#### git commit -m '変更内容'

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

#### ローカルでmergeすると個人で確認を挟まずにリモートへアップロードすることができるが、プルリクエストでmergeするとgit hub上でmergeする前にプルリクエストを作成してリーダーが確認の上mergeする

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

### 状況確認をしてどれかの変更を1つ残すかもしくはコンフリクトを起こしてしまった変更点を組み合わせて修正する。
#### 1. 先にマージされた変更内容を取り込む
#### 2. 後にマージしようとしている変更内容を取り込む
#### 3. どちらの変更内容も取り込む
